# Board Game Chooser

This is a simple React / Express project that attempts to use all best practices, with the intention to be used for a tutorial series. Please contact the author (EdSalisbury) if you're interested in seeing a tutorial series on this.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- Node.js
- Yarn (or npm)

### Installing

First, clone the repository to your local machine:

```bash
git clone https://github.com/EdSalisbury/bgchooser
```

Then, navigate to the project directory and install the dependencies:

```bash
cd bgchooser
yarn install
```

### Running the Server

To start the server, run:

```bash
yarn start
```

The server will start on port 3000. You can change this by modifying the `port` constant in `server.ts`.
